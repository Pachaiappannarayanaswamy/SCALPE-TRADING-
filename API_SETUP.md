# AI Analysis Setup Guide

## Getting Your Free Gemini API Key

1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy your API key
5. Go to the AI Analysis page in SCALPE
6. Paste your API key in the input field and click "Save Key"

## Using the AI Analysis Feature

1. Navigate to the **AI Analysis** page from the main navigation
2. Enter and save your Gemini API key (first time only)
3. Upload a trading chart image by:
   - Clicking the upload zone, OR
   - Dragging and dropping an image file
4. Click **"Analyze with AI"** to get instant trading insights
5. View your analysis history below

## Supported Image Formats

- PNG
- JPG/JPEG
- WEBP
- Maximum file size: 10MB

## Features

- **Pattern Recognition**: Identifies chart patterns (head & shoulders, triangles, flags, etc.)
- **Trend Analysis**: Determines bullish, bearish, or neutral trends
- **Support/Resistance**: Identifies key price levels
- **Entry/Exit Points**: Suggests optimal trading positions
- **Risk Assessment**: Provides risk analysis
- **Trading Recommendations**: Actionable trading advice

## Privacy

- API keys are stored locally in your browser
- Images are processed by Google Gemini API
- Analysis history is stored locally (last 10 analyses)
